**what currently exists?**
A comprehensive, full-stack investment analysis application branded as the Wheeler Family Portfolio. It uses a FastAPI backend and a React frontend (with TailwindCSS and Shadcn UI). The application operates without user authentication, loading with a pre-populated dummy portfolio that covers multiple individuals (James and Sarah Wheeler), companies, and trust structures.

The UI features a left-hand sidebar for navigation. It includes a wide array of calculators and tools:
- **Dashboard**: Portfolio summary, asset allocation, and investment projections.
- **Calculators**: Tax Analysis, Loan, Monte Carlo, Capital Gains (CGT), Salary Packaging, Dividend Reinvestment, Division 7A, and CGT Event Tracker.
- **Planning Tools**: Scenario Builder (supporting personal, company, and trust entities), Household Budget (with weekly/fortnightly/monthly frequencies), Property Comparison, Scenario Comparison, Income Splitting, and more.
- **Reporting**: A dynamic Report Generator that provides an executive summary, data breakdowns, and AI-driven Recommended Next Steps based on the user's portfolio data.
